Pourquoi mon lien ne marche pas ?

Certains sites empêchent les navigateurs d’accéder directement à leur contenu pour des raisons de sécurité. C’est ce qu’on appelle la politique CORS.

À cause de cela, mon scanner ne peut pas analyser automatiquement certaines pages via leur URL. Ce n’est pas un bug : c’est une protection imposée par le site que vous essayez d’analyser.

Comment contourner ce problème ?

Vous pouvez toujours analyser la page en suivant cette méthode :

  1. Ouvrez la page que vous voulez analyser
  2. Faites clic droit → Afficher le code source
  3. Ou utilisez le raccourci CTRL + U (Windows) ou CMD + Option + U (Mac)
  4. Copiez tout le code HTML
  5. Collez-le dans la zone “Coller le code HTML” sur la page d’accueil

Votre analyse fonctionnera alors à 100%, même si le site bloque les requêtes.
