À propos du projet

Ce projet a été réalisé dans un cadre pédagogique afin de mettre en pratique des notions de développement web, de cybersécurité et d’accessibilité.

Objectifs

  • Sensibiliser aux risques liés à la sécurité web
  • Présenter les principes de l’accessibilité web
  • Proposer une analyse simple et compréhensible

Technologies utilisées

  • HTML / CSS
  • JavaScript
  • Hébergement sur Vercel

Aucune donnée personnelle n’est collectée ou stockée.
